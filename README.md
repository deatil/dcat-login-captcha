## 预览

![LoginCaptcha](https://user-images.githubusercontent.com/24578855/109408841-53b93980-79c8-11eb-8293-f41b629838a5.jpg)


## 环境
 - PHP >= 7.2.5
 - Dcat-admin ^2.0


## 安装

### composer 安装

```
composer require lake/login-captcha
```

### 安装扩展

在 `开发工具->扩展` 安装本扩展


### 开源协议

*  本扩展 遵循 `Apache2` 开源协议发布，在保留本扩展版权的情况下提供个人及商业免费使用。
